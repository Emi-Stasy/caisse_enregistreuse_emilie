# caisse_enregistreuse_emilie

## Description:
Simule une caisse enregistreuse en roupies Mauricienne(Rs).
Le programme genere un montant aleatoire et demande le paiement

##Compiler
gcc -o caisse caisse.c

## Executer
.\caisse

## Utilisation
- Le programme affiche le montant a payer
- Entrez le montant donne par le client
